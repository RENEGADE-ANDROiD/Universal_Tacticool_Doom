# Universal Tacticool Doom (Doom II/Heretic/Hexen)
Universal zscript addon that adds Weapon Lowering, Rotational Tilting and Weapon Lean movement.
------------------
New code with more dependable weapon lowering and rotational tilting! 

------------------
An experimental fork of 'Universal-Weapon-Tilt-GZ' by Generic Name Guy.
This branch adds code made by Dithered Output, maker of Siren, that has all been heavily modified. Also included is a trimmed down version of Immerse Lean that includes modified to allow you to move slowly while peeking around corners. Lean code written by Joshua Hard (josh771).

Includes a customized menu with slider options for everything.
See Control Options to set Lean Keybind
------------------
Does NOT include Tilt++ or Weapon Sway.

Works universally for Doom, Doom II, Heretic and Hexen.
